운동관리 PWA v2 자유순서 버전

1. GitHub의 운동관리 저장소를 엽니다.
2. Add file → Upload files를 선택합니다.
3. 이 폴더 안의 파일 전체를 저장소 최상위에 업로드합니다.
   - index.html
   - manifest.webmanifest
   - service-worker.js
   - icons 폴더
4. 같은 이름의 기존 파일을 덮어쓰고 Commit changes를 누릅니다.
5. GitHub Pages 배포가 완료될 때까지 잠시 기다립니다.
6. 아이폰 Safari에서 기존 Pages 주소를 열고 새로고침합니다.
7. 홈 화면 앱을 완전히 종료한 뒤 다시 실행합니다.

주의
- Safari 웹사이트 데이터는 삭제하지 마세요. 기존 localStorage 기록이 사라질 수 있습니다.
- 기존 v3 기록은 가능한 범위에서 읽어오지만, 새 루틴에 없는 운동 기록은 통계에 표시되지 않을 수 있습니다.
- LEGS에서는 그날 사용할 스쿼트를 핵스쿼트 또는 스미스 스쿼트(둔근) 중 선택합니다.
