운동관리 PWA 오프라인 버전 — 배포 및 설치

[구성 파일]
- index.html
- manifest.webmanifest
- service-worker.js
- icons/ 폴더

[GitHub Pages 배포]
1. 위 파일과 icons 폴더를 GitHub Pages 저장소의 같은 경로에 모두 업로드합니다.
2. GitHub Pages 주소를 아이폰 Safari에서 온라인 상태로 한 번 엽니다.
3. 첫 화면이 완전히 열린 뒤 5~10초 정도 기다립니다. 이때 오프라인용 파일이 저장됩니다.
4. Safari 공유 버튼 → '홈 화면에 추가'를 선택합니다.
5. 홈 화면의 '운동관리' 아이콘으로 실행합니다.

[오프라인 확인]
- 온라인 상태에서 한 번 정상 실행한 다음 비행기 모드로 전환합니다.
- 홈 화면 아이콘을 다시 열어 운동 화면이 표시되면 정상입니다.
- 최초 접속 전에는 오프라인 실행이 불가능합니다.

[데이터 저장]
- 운동 및 체중 기록은 아이폰의 localStorage에 저장됩니다.
- 인터넷 연결과 무관하게 기록할 수 있습니다.
- Safari 웹사이트 데이터 또는 홈 화면 앱 데이터를 삭제하면 기록이 사라질 수 있습니다.
- 통계 → 데이터 관리 → JSON 백업을 정기적으로 실행하십시오.

[업데이트 시 주의]
- 새 버전 파일을 GitHub Pages에 덮어쓴 뒤 온라인 상태에서 앱을 한 번 다시 실행합니다.
- 화면이 예전 버전이면 앱을 완전히 종료하고 다시 열거나 Safari에서 페이지를 새로고침합니다.

[버전 1.1 변경사항]
- 아이폰 운동 화면 하단에 '이전 / 저장 후 다음 / 다음' 조작부를 고정했습니다.
- 화면을 아래로 스크롤하지 않아도 기록 저장과 운동 이동이 가능합니다.
- 기존 localStorage 키를 유지하므로 같은 주소에 덮어쓰면 기존 운동 기록이 그대로 이어집니다.
- GitHub에 새 파일을 올린 뒤 홈 화면 앱을 완전히 종료하고 다시 실행하세요.
